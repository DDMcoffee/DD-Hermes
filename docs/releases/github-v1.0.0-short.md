# DD Hermes v1.0.0 Short Release Copy

Use this file as the compact GitHub Release body for the first public DD Hermes release.

## Release Title

```text
DD Hermes v1.0.0
```

## Short GitHub Release Body

### English

```md
DD Hermes is a workspace-first engineering harness for complex agent work.

This V1 release publishes the reusable public surface of the project:

- explicit task artifacts such as `contract`, `state`, `context`, `handoff`, and `closeout`
- scripts and hooks for workflow control, verification, and guardrails
- cross-repo execution guidance for keeping control-plane work separate from target business code
- MIT licensing for public reuse

Start here:

1. `README.md`
2. `docs/quickstart.md`
3. `./scripts/demo-entry.sh`

Minimum verification:

```bash
bash tests/smoke.sh schema
./scripts/demo-entry.sh
```
```
